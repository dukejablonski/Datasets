BBC Planet Earth Dataset for Scene Detection
============================================

This dataset features eleven episodes from the BBC Educational TV series Planet Earth.

File contents:
- annotations/shots/<episode>.txt files contain ground truth shots. Each row contains the starting and ending frames of a shot.
- annotations/scenes/<annotator>/<episode>.txt files contain ground truth scenes of a given annotator, expressed as a comma separated list of shot indexes. Each index represents the starting shot of a new scene.

Video files can be found at: https://www.mediafire.com/folder/texdwptt9242j/BBCPH
